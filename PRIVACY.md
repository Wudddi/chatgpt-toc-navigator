# Privacy Policy

ChatGPT TOC Navigator does not collect, store, transmit, or sell personal data.

## What the extension does
- It reads the content of the ChatGPT webpage locally in your browser to build an on-page table of contents (TOC).
- It stores only UI preferences locally (e.g., panel position, minimized state) using browser storage.

## Data collection
- No data is sent to any external server.
- No analytics or tracking is performed.

## Contact
If you have questions, please open an issue in the GitHub repository.
